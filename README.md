## üõ†Ô∏è Setup Instructions

To enable commit statistics in this README:
1. Create a `.github/workflows/commit-stats.yml` file in your `nitinneyoo` repository with the workflow below.
2. Ensure your repository is public for stats to display.
3. Verify your Git email (`nitinsinghji12@gmail.com`) is linked to your GitHub account to track commits correctly.

```yaml
name: Update Commit Stats
on:
  push:
  schedule:
    - cron: '0 0 * * *' # Runs daily at midnight UTC
jobs:
  update-stats:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.x'
      - name: Install dependencies
        run: pip install pygit2
      - name: Generate commit stats
        run: |
          python -c "
          import pygit2
          from datetime import datetime, timedelta
          import matplotlib.pyplot as plt
          from collections import Counter
          import os

          repo = pygit2.Repository('.')
          commits = []
          for commit in repo.walk(repo.head.target):
              commit_time = datetime.fromtimestamp(commit.commit_time)
              commits.append(commit_time)

          # Yearly commits
          yearly = Counter(c.year for c in commits)
          plt.bar(yearly.keys(), yearly.values())
          plt.title('Yearly Commits')
          plt.savefig('commits-yearly.svg')
          plt.close()

          # Monthly commits (2025)
          monthly = Counter(c.strftime('%Y-%m') for c in commits if c.year == 2025)
          plt.bar(monthly.keys(), monthly.values())
          plt.title('Monthly Commits (2025)')
          plt.xticks(rotation=45)
          plt.savefig('commits-monthly-2025.svg')
          plt.close()

          # Daily commits (last 30 days)
          start_date = datetime.now() - timedelta(days=30)
          daily = Counter(c.strftime('%Y-%m-%d') for c in commits if c >= start_date)
          plt.bar(daily.keys(), daily.values())
          plt.title('Daily Commits (Last 30 Days)')
          plt.xticks(rotation=45)
          plt.savefig('commits-daily.svg')
          plt.close()
          "
      - name: Commit updated stats
        run: |
          git config user.name 'github-actions'
          git config user.email 'actions@github.com'
          git add commits-*.svg
          git commit -m 'Update commit stats'
          git push
